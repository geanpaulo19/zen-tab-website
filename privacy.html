Zen New Tab - Privacy Policy

Effective Date: January 16, 2026

Commitment to Privacy: Zen New Tab is designed to be a private-by-default, minimalist dashboard. We do not track you, we do not sell your data, and we do not collect any personally identifiable information (PII).

1. Single Purpose & Description

Zen New Tab provides a unified minimalist dashboard that streamlines your start-page experience. It harmonizes essential elements like time, date, daily inspiration, weather, and integrated search into one cohesive interface to eliminate digital clutter.

2. Data Collection and Limited Use

This extension does not collect personal data. All features operate locally within your browser. Zen New Tab adheres to the Google Web Store Limited Use Policy:

  • We do not use or transfer data for serving ads, including retargeting or personalized advertising.
  • We do not allow humans to read user data unless we have first obtained your affirmative agreement for specific purposes (e.g., tech support).

3. API Usage & Search Functionality

The extension fetches dynamic content to power the dashboard via secure (HTTPS) requests:

  • Backgrounds: Retrieves high-quality imagery via proxy from Unsplash, and Lorem Picsum.
  • Inspiration: Fetches daily quotes from ZenQuotes, TypeFit, and Quotes-DB.
  • Weather: Uses the Open-Meteo API to provide real-time updates based on your location.
  • Search Suggestions: When you type in the integrated search bar, your query is sent to our Search API solely to provide real-time suggestions. This data is processed in transit, is never stored, and is not linked to your identity.

4. Geolocation Data

Geolocation is used exclusively to provide local weather information. The location data is processed locally to determine your city and is never shared, stored, or used for any other purpose.

5. Local Storage

We use the browser's local storage to store your preferences locally on your device, including:

  • UI preferences (12/24-hour time, units).
  • Cached background and quote data for faster loading.
  • Temporary cache for weather and search suggestions.

6. No Remote Code Execution

In compliance with Chrome Web Store safety standards, Zen New Tab does not execute remote code. All APIs return structured JSON data only. No scripts, eval(), or executable content is ever fetched from external sources.

7. Third-Party Services

If you click on photographer credits, you may be directed to Unsplash. We recommend reviewing their respective privacy policy.
