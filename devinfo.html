What's new

Changelog

Every update, improvement, and new feature — documented as they ship.

v2.5.1 July 04, 2026 Latest
  • Appearance customization — added curated theme presets, wallpaper overlay strength, glass intensity, clock style, search bar style, quick-link tile style, and text shadow controls.
  • Added four new Theme Presets: Ocean Mist, Plum Dusk, Rose Quartz, and Slate Mint.
  • Expanded Appearance controls with widget corners, search width, clock size, quick-link size, UI scale, background blur, widget shadow, and font style presets.
  • Added an Appearance Reset action to restore theme, accent, quick-link shape, and visual controls back to defaults.
  • Quick Link Icon Shape now includes Rounded, Soft, Square, Circle, Arch, and Pebble options with smoother live preview transitions.
  • Theme presets and accent colors now work independently, so changing a theme does not overwrite the user's selected accent color.
  • Search bar placeholder layout was refined with split text/icon alignment and a cleaner centered magnifier mark.
  • Public Holidays — removed unsupported countries from the picker and improved Auto-detect so it uses timezone before browser language.
  • Public Holidays cache writes are now safer, so storage quota issues no longer make the widget look broken.
  • Countdown Timer now clamps invalid, blank, or overflowing minute/second inputs instead of rendering NaN:NaN.
  • Event Countdown now defaults to the user's local date instead of a UTC date, preventing off-by-one-day behavior in some timezones.
  • Pomodoro restored sessions are sanitized, and pressing play from 00:00 starts a fresh session.
  • 2048, Memory, Snake, and Tetris now reject malformed saved game state instead of restoring corrupted boards.
  • Snake now handles a completely filled board without getting stuck trying to spawn food forever.
  • Tetris paused state now behaves as labeled: random keys no longer resume the game.
  • World Clock, To-do, Event Countdown, and Focus Sounds now normalize saved local data before rendering.
  • Quick Links now reject unsafe URL schemes and normalize missing protocols more reliably.
  • Search suggestions, quote rendering, weather caching, and hidden widget timers received additional reliability fixes.
v2.2 June 11, 2026
  • Mini Apps card grid — Settings now shows each mini app as a tappable card with icon, description, and inline toggle. Easier to scan than the old list.
  • Added 🧮 Calculator — full arithmetic with percentage, sign toggle, and keyboard support.
  • Added 📊 Day Progress — animated progress bars for day, week, and year elapsed.
  • Added 🧩 2048 — classic sliding number puzzle.
  • Added 🃏 Memory Match — flip cards to find matching pairs.
  • Added 🐍 Snake — classic snake game with keyboard controls.
  • Added 🧱 Tetris — stack and clear lines with falling blocks.
  • Added 📆 Event Countdown — count down to your own custom dates and events.
  • Added 🎉 Public Holidays — upcoming holidays for your selected country.
  • Added 🫁 Breathing Exercise — guided deep breathing with animated circle indicator.
  • All mini apps now freely draggable and repositionable anywhere on screen.
  • Performance: consolidated all widget event listeners into a single shared drag handler.
  • Bug fixes and polish across all mini apps and the settings modal.
v2.1 March 28, 2026
  • Introduced quick links favicon support — auto-fetch site icons for each bookmark.
  • Added 12/24-hour time format toggle with system-auto detection.
  • Weather refresh now detects 1 km location changes for smarter API calls.
  • Background sources shuffle between APIs (Unsplash, Picsum) for greater variety.
  • Stability and caching improvements across weather, quote, and background fetchers.
v2.0.0 March 21, 2026
  • Complete rewrite — unified all widgets under a single Settings Modal (⚙️) with toggle switches, replacing scattered floating buttons.
  • Added 🍅 Pomodoro Timer with Focus, Short Break, and Long Break modes, animated SVG ring, and wall-clock accuracy across tab switches.
  • Added 🌍 World Clock — up to 8 cities with live local times; add any city by name with automatic IANA timezone lookup via Nominatim and Open-Meteo.
  • Added ✅ To-do List — draggable checklist widget with persistent storage, done/undone toggle, and one-click clear of completed items.
  • Added ⏱ Countdown Timer with 5 quick-presets (5m to 1h), custom input, animated SVG ring, tab-title countdown, and audio alarm on finish.
  • Added 📅 Calendar — draggable monthly view with today highlighted, weekend dimming, and month navigation.
  • Added 🎵 Focus Sounds — offline ambient noise (Rain, Wind, Ocean, Fire, White Noise, Brown Noise) generated entirely via Web Audio API with volume control.
  • Added 🖼️ Custom Wallpaper — upload your own image, stored locally in the browser, independent from the daily photo feed.
  • Added ✨ Particle system — subtle floating particles overlaid on the background, toggleable in Settings.
  • Added Quick Links manager in Settings — add, edit, and remove links inline without leaving the modal; max 8 links enforced.
  • Introduced a shared drag system — one global mousemove/mouseup handler for all widgets, eliminating per-widget event listener leaks.
  • Weather now uses a 15-minute TTL cache with 1 km location-change detection to avoid redundant network fetches.
  • Background fetcher shuffles API sources (Unsplash Worker vs. Lorem Picsum) on each load for greater image variety.
  • All timers use wall-clock anchoring via Date.now() so they stay accurate when the tab is backgrounded.
  • 24-hour detection cached once at startup. Entrance animations skip hidden elements. Ghost-text layout values cached after first measurement.
v1.3.0 March 01, 2026
  • Introduced 📝 Vibrant Sticky Notes with heavy glassmorphism effects.
  • Random jewel-toned frosted glass color system generated on each note creation.
  • Persistent storage for notes: content, screen position, and custom colors all saved between sessions.
  • Enhanced backdrop-filter blur (25px) for a deeper, more premium frosted aesthetic.
  • Improved placeholder readability with adjusted contrast for dark backgrounds.
v1.2 January 16, 2026
  • Added 🌤️ subtle weather widget using local geolocation data.
  • Integrated 🔍 streamlined search bar with real-time suggestions.
  • Smoother transitions and responsive scaling throughout the interface.
  • Refined placeholder text for a more minimalist feel.
v1.1 January 05, 2026
  • Optimized main time display for better desktop readability.
  • Implemented breathing container effects for responsive resizing.
  • Improved element entrance animations using cubic-bezier curves.
v1.0 January 02, 2026 Initial release
  • Initial public release of the Zen New Tab experience.
  • 12/24-hour clock toggle and daily inspiration quotes.
  • Multi-source background imagery from Unsplash, Pexels, and Picsum.
